Array Methods

Here is a list of all array methods

  • Filter Method:

    Iterates an array and performs a check for each function inside the array.

  • Map Method:

    This creates a new array based on the current array.

  • Reduce Method:

    Returns any single value based on the arrays listed.

    Or in simpler terms, returns the selected arrays.

  • Find Method:

    This method returns the first value in the array that passes a certain test in a callback function.

    Or rather, it just picks the first value that corresponds to the criteria.

  • Sort Method:

    This method sorts the array in a way the user feels like, either from shortest to longest, etc.

  • Chaining Arrays Method:

    This involves using multiple array methods at once.
